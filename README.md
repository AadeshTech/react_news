 A Reactjs web App to get daily news including categories.

## Contributing

## Tech Stack

**Client:** React ,Bootstrap.


## Run Locally

Clone the project

```bash
  git clone https://github.com/0Pixel0/news-monk
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
